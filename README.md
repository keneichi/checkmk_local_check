# checkmk_local_check
Ce repo fourni des check locaux installable sur les hosts afin d'affiner  le monitoring.  

Les checks sont répartis par sytème d'exploitation (Windows / Linux).
Chaque check voit son utilité définie dans le fichier Readme.md  
Si plusieurs fichiers sont nécessaires (ex : fichier de configuration) un sous dossier portant le nom du check est présent.  
Dans ce cas des instruction supplémentaire sont données dans le fichier readme.md du dossier en question

**Ces scripts nécéssitent, bien évidemment, que l'agent checkmk soit installé.**
