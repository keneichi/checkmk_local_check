# Généralités
Sauf mention contraire, tous les check locaux sont à installer dans le dossier **C:\ProgramData\checkmk\agent\local**

## windows_updates_and_reboot
Deux check sont créés.  
* Un premier qui vérifie la présence de mises à jour disponible pour Windows
* Un second qui vérifie si un reboot est nécessaire après une mise à jour.
